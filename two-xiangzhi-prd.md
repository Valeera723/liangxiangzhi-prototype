# 两相知：抖音小程序创意方案

## 1. 产品定位

**两相知** 是一个面向抖音情感、关系、职场内容场景的小程序功能。它不做“面相断命”或“人像评分”，而是先用《神相全编》《麻衣神相》《柳庄相法》《冰鉴》等古籍中的“神、骨、形、气、色、眉势、鼻准、兰台廷尉、地阁”等术语，对两张照片做非贬低的相貌象意解析，再结合关系类型和当前卡点生成一份“古籍风格合拍签”。

一句话：**先观双相，再问关系，看看你们的矛盾卡在什么节候。**

核心价值不是替用户决定一段关系，而是在情绪上头、冷战、误会、暧昧拉扯、合作不顺时，给用户一个轻量、有仪式感、可分享的关系整理入口。

## 2. 合规边界

必须坚持以下原则：

- 用户主动进入，不基于平台私下推断情绪状态做强打扰弹窗。
- 用户主动上传照片，并明确确认已获得对方同意。
- 不做身份识别、人脸库比对、颜值评分、人格定论、命运判断。
- 面相分析只描述眉毛走势、鼻峰鼻翼、下颚线、目神、气色氛围等“相学象意”，不对个人做贬低。
- 不输出“必须分手”“对方克你”“此人不可信”等伤害性或决定性结论。
- 不长期保存照片；默认仅用于本次生成。
- 结果标注为古籍文化风格娱乐内容，不构成心理、医疗、法律或人生决策建议。

推荐对外表述：

> 本功能为古籍文化风格的娱乐化关系报告。照片仅用于本次生成，不用于身份识别、人脸库比对或长期保存。

## 3. 目标用户

- 情绪低落、反复刷情感内容的人。
- 情侣、暧昧对象之间出现误会或冷战的人。
- 上下级、伙伴、同事（同门）之间沟通或合作不顺的人。
- 对明星/公众人物关系产生强烈情绪投射、需要边界整理的人。
- 喜欢玄学、国风、心理测试、关系测试内容的年轻用户。

## 4. 核心场景

### 场景 A：情感视频挂载

用户刷到“冷战后该不该主动联系”“暧昧对象突然冷淡”等视频，视频挂载小程序卡片：

**别再内耗了，让古法合拍签看看你们卡在哪里。**

### 场景 B：评论区裂变

评论区引导：

- “情侣选冷战，看看你们是哪种合拍签。”
- “上下级、同门、伙伴也能测，重点看边界和节奏。”
- “不是算分手，是给一句能开口的话。”

### 场景 C：分享卡传播

生成结果可分享为短图卡：

- 我和 TA 是「山泽互照签」
- 合拍势能 82
- 今日破冰话：我不是想争输赢，我只是想知道，我们可不可以重新站到同一边。

## 5. 用户路径

1. 进入小程序。
2. 开场展示水墨面相古图与古籍引文，文字竖排、从右往左展开，进入“启卷”仪式。
3. 上传两张照片并勾选授权确认。
4. 等待“相法推演”动态判断：定三庭、察五官、参气色、合双相。
5. 先生成“甲相 / 乙相”古法面相术语解析。
6. 在面相初诊后询问：“您是否想通过回答几个问题，获得更加针对性的建议？”
7. 再询问你们现在是什么关系，只保留六类：情侣、上下级、伙伴、明星、暧昧、同事（同门）。
8. 动画翻卷进入下一界面，让用户自由输入你们之间遇到的问题，以及提问者最想得到的建议。
9. 若用户愿意问诊，则按关系类型回答 5-7 个常见选择题。
10. 将双相面相摘要、关系类型、问题自述、需求和问诊回答交给 DeepSeek 代理合参成签。
11. 分享结果或重新生成。

## 6. 报告结构

### 6.0 双相小记

先分别分析两个人的面相象意。术语要专业、有古籍气质，但不能贬低个人。

可用分析维度：

- 眉势：眉形走势、眉尾舒展、眉心聚散。
- 目神：眼神聚散、神光含露、神藏不露。
- 鼻峰：山根、鼻梁、鼻准形态。
- 鼻翼：兰台、廷尉开合。
- 颐颌：下颚线、地阁、下停走势。
- 气色：照片氛围、神态松紧，只做当下氛围描述。

原型阶段判断方式：

- 接入浏览器本地面部点阵模型，定位眉、眼、鼻、口、颌等结构关键点。
- 计算三庭比例、眉尾走向、眉弓起伏、眉眼距、眼裂开合、鼻梁与鼻翼比例、口角平衡、下颌线与地阁居中等信号。
- 同时用画布读取照片明暗、对称、边缘密度、中心明度，作为气色氛围与轮廓辅助。
- 用规则映射到相学术语，例如“眉尾舒扬”“目宫开阔”“山根有承”“兰台廷尉收放相齐”“地阁正承”。
- 等待页展示双相取点台：自拍图上叠加面部点阵、五官连线、三庭参考线与水墨扫描层；中央设置“AI 复核镜”，让甲相和乙相带关键点交替放大呈现，并至少停留约 9 秒完成多轮复核，形成古今结合的判断过程感。
- 分析页展示“实测入相”：把三庭、眉势、目神、鼻准、兰台、地阁等测量分值与对应古法术语并排显示。
- 分析页新增“F-BTI”：把每个人归为嘴替书生、慢热高僧、排班谋士、憋招军师、硬控将军、秒懂剑客、走神行者、边界镖客等温和类型，并配一位历史人物水墨头像作为“象意参照”，不做真实人物断言。
- 甲乙合参新增类型关系：同频递茶局、开合互译局、快慢搭台局、眉目互照局、山泽相承局，不使用相克、克制、命定等说法。
- 用户只选择面相快速合参时，不调用大模型；用户选择回答问诊题后，才将面相摘要、F-BTI、类型关系、问题与选择题回答交给合参代理。
- 如果模型网络不可用，原型会明确降级为离线区位估算，不伪装成精细识别。

正式上线建议：

- 保持端侧识别优先，照片不上传服务器；如需云端能力，必须单独授权。
- 仍然只输出非身份化摘要，不保存原始照片、人脸模板或可还原身份的人脸特征。
- 输出必须保持“象意/倾向”语气，不做真实人格定论。

示例：

- 眉势清扬，眉尾略舒，古称清扬入鬓之象；象意为做事先立章法，情绪起时仍重体面。
- 山根承起，鼻峰不逼，准头有容；象意为遇事多先权衡轻重，不喜仓促表态。
- 颐线柔中有势，口角气息收敛；象意为表达未必外放，但心中会反复衡量对方态度。

### 6.1 合拍签名

以古法意象命名，避免直接说“好坏”。

示例：

- 山泽互照签：稳定与回应的节奏互补。
- 水木相生签：高共情、适合慢谈。
- 风火相激签：强吸引，也容易冲撞。
- 金水相照签：原则与弹性需要重新对齐。

### 6.2 合拍势能

用“势能”代替“命中注定”或“适配度”。分数只做娱乐化表达。

不建议低于 60，避免制造焦虑。建议区间：72-92。

### 6.3 同频点

描述关系中值得保留的连接方式。

### 6.4 内耗点

描述沟通错位，不指责任何一方。

### 6.5 破冰话

给用户一句可直接发给对方的话，这是产品最实用的部分。

### 6.6 三日小任务

给一个轻量行动，不要求用户马上做重大决定。

## 7. 古籍概念转译

| 古籍风格概念 | 现代关系维度 | 输出方式 |
| --- | --- | --- |
| 神 | 注意力、回应感、眼神状态 | “目神有聚，象在回应需求较强” |
| 骨 | 稳定度、边界感、承托感 | “颐颌有托，象在重承诺与秩序” |
| 形 | 行动节奏、表达路径 | “眉势顺生，象在先理后情” |
| 气 | 冲突气氛、表达温度 | “气息偏收，宜先安神再论事” |
| 色 | 当下状态、照片氛围 | 只做氛围描述，不做健康判断 |
| 兰台廷尉 | 鼻翼开合、现实安全感 | “在关系里重实际行动与确认” |
| 地阁 | 下停、下颚线、长期承托 | “更看重稳定边界和长期兑现” |

## 8. 生成逻辑

### 8.1 输入

- face_analysis_a：甲相术语解析。
- face_analysis_b：乙相术语解析。
- relation_type：关系类型，在面相解析后询问。
- problem_text：用户自由输入的当前问题。
- need_text：提问者最想得到的建议或需求。
- consultation_answers：按关系类型生成的 5-7 个选择题问诊回答。
- image_quality：照片质量、清晰度、光线。
- face_landmark_summary：五官定位摘要，只用于风格化维度，不存储身份特征。
- safety_flags：未成年人、多人照片、遮挡严重、疑似非真人等。

### 8.2 中间层

将照片与情境转成非身份化标签：

- rhythm：快节奏 / 慢节奏 / 一快一慢。
- expression_symbol：外放 / 内收 / 克制。
- stability_symbol：稳定型 / 变化型。
- conflict_style_symbol：追问与回避 / 先理后情 / 先情后理。
- relation_need_symbol：被回应 / 被尊重 / 被确认 / 被理解。

### 8.3 输出约束

生成模型必须遵守：

- 不判断人格好坏。
- 不判断忠诚、出轨、疾病、寿命、财运。
- 不输出不可逆关系建议。
- 所有建议必须可执行、低风险、非攻击性。

## 9. MVP 范围

第一版只做：

- 双人照片上传。
- 授权确认。
- 双相小记。
- 关系类型选择。
- 面相初诊后问诊选择。
- 自由输入关系卡点与提问需求。
- 按关系类型生成 5-7 个选择题问诊问题。
- DeepSeek 代理合参成签。
- 合拍签报告。
- 分享卡。

暂不做：

- 长期关系档案。
- 每日推送。
- 真人咨询。
- 复杂社交关系图谱。

## 11. 内容安全词库

禁止输出：

- “你们必分”
- “此人克你”
- “对方不可信”
- “对方会背叛”
- “面相不好”
- “命不好”
- “旺夫”“克夫”等性别刻板结论

推荐表达：

- “你们目前的节奏需要重新对齐。”
- “这段关系不是不能继续，而是需要更清楚地表达边界。”
- “先处理情绪，再处理问题。”
- “不要把沉默自动理解为否定。”

## 12. 可直接使用的推广文案

- 别再内耗了，抽一张你和 TA 的合拍签。
- 不是算分手，是帮你找到开口的方式。
- 有些关系不是不合，是说话顺序错了。
- 冷战第几天了？看看你们卡在了哪一步。
- 上下级、同门、伙伴也能测，重点看边界、节奏和信任。
